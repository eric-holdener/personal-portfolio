Facebook Clone

With this project, I set out to create a functioning clone of Facebook, trying to replicate the core features that turned it into the social media giant it is today. This was the capstone project to TheOdinProjects Ruby on Rails curriculum, and allowed me to create a fully functioning app with very little guidance on the part of my learning module. Below you can see some key features I managed to implement, as well as the technology used in the creation. If you want, you can also check out the app for yourself, at the links below. They will take you to the website and to my github repo, where I go more into depth on what I learned and what I want to improve in the future.

Core Features

  • Adding and deleting friends
  • Creating posts with just text or adding images as well
  • User profiles and authentication
  • Image database hosted on a bucket instance of AWS
  • Liking and disliking posts
  • Leaving comments

Tech Stack Utilized

  • Ruby (and Ruby gems, like Devise)
  • Ruby on Rails
  • Postgre SQL
  • Boostrap
  • AWS
  • Heroku

Twitter Sentiment Analysis

I actually minored in Data Analytics while in college, and as a thought experiment I wanted to try and put together a sentiment analysis bot that could analyze a number of tweets about a given topic and see what people were thinking about it at the time. I actually proved to myself that it worked properly by running it on a sample of 10,000 tweets about Will Smith right after his incident with Chris Rock, and it returned over 50% negative, which I felt was right. If you want to see more about my code, please check out the link to its Github repo below! Unfortunately it functions more as a script than anything so I haven't deployed it, but I would be more than happy to give a demonstration if you are interested! Just find my details in the contact me page down below.

Key Takeaways and Learnings

  • Naive Bayes Theory
  • Bag of Words processing
  • Creating appealing graphs with Matplotlib
  • Pandas data processing
  • Web scraping and the limitations of scraping social media sites

Tech Stack Utilized

  • Python
  • Pandas
  • NLTK
  • Numpy
  • Twint
  • Matplotlib
